## 🏗️ 技术架构

- **样式**: Tailwind CSS
- **状态管理**: React Hooks

## 🚀 部署选项

### 1. 一键部署（推荐）
```bash
# 使用Vercel
npm install -g vercel
vercel --prod

# 或使用GitHub Pages
# 推送代码后，在GitHub设置中启用Pages
```

### 2. 手动部署
- **Vercel**: 拖拽项目文件夹
- **Netlify**: 拖拽项目文件夹  
- **GitHub Pages**: 上传到仓库并启用

### 3. 本地开发
```bash
# 克隆项目
git clone https://github.com/tengfeizhao1219/No-Words.git

# 安装依赖
cd No-Words && npm install

# 启动开发服务器
npm run dev
```
