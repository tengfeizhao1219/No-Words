## 🎯 产品定位

**产品名称**: 不记单词  
**Slogan**: 查询即收藏，无感积累  
**定位**: 个人使用的单词查询翻译 + 复习记忆工具  
**形态**: PWA (渐进式 Web 应用)  
**核心理念**: 
- 不刻意去"记"，而是自然积累
- 免安装、即用即走
- 简洁无干扰

---

## 😫 核心痛点

1. 生活/工作中遇到陌生单词/句子时，需要统一的查询入口
2. 查过的单词容易忘记，没有自动记录机制
3. 后续想复习时，找不到曾经查过的内容
4. 现有工具功能繁杂，不够简洁

---

## 📱 产品结构（MVP）

### 3 个核心页面

| 页面 | 功能 | 路由 |
|------|------|------|
| **首页** | 大搜索框 + 翻译结果（自动保存）+ 收藏按钮 | `/` |
| **词库** | 历史记录/已收藏列表（卡片式） | `/library` |
| **复习** | 闪卡模式界面 | `/review` |

### 底部导航栏

```
[首页]  [词库]  [复习]
```

## 📄 页面详细设计

### 1️⃣ 首页 (`/`)

**核心交互流程**:
1. 打开 App → 大搜索框
2. 输入单词/句子 → 自动显示翻译结果
3. 结果下方显示「收藏」按钮（可取消）
4. 查询记录自动保存到词库

**页面元素**:
- 顶部：搜索框（带语音输入图标）
- 中部：翻译结果卡片（英文 + 中文 + 发音按钮）
- 底部：收藏按钮（默认选中）

**技术要点**:
- 输入防抖（300ms）
- 自动翻译
- 查询后自动保存（用户可取消收藏）

---

### 2️⃣ 词库页 (`/library`)

**页面设计**:
- **展示形式**: 卡片式布局
- **排序**: 按时间倒序（最新的在前）
- **卡片内容**: 单词 + 中文释义 + 查询日期
- **交互**: 点击卡片查看详情/编辑

**功能**:
- 搜索已收藏单词
- 删除单个记录
- 批量操作（后续迭代）

---

### 3️⃣ 复习页 (`/review`)

**闪卡模式设计**:
- **卡片展示**: 默认英文朝上，点击翻面显示中文
- **掌握标记**: 翻面后显示两个按钮
  - ✅ 掌握了 → 下一张
  - ❌ 没记住 → 加入待复习队列
- **进度显示**: 顶部显示当前进度（如 5/20）

**复习模式**:
- 全部单词复习
- 待复习队列（标记为"没记住"的）

## 🛠️ 技术架构

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| **框架** | React + TypeScript | 主流、生态好 |

| **短信验证码** | 可选（初期用邮箱） | ~0.04 元/条 |

## 🗄️ 数据库设计

### 用户表 (`users`)
```sql
- id (UUID, 主键)
- email (字符串，唯一)
- phone (字符串，可选)
- created_at (时间戳)
```

### 单词表 (`words`)
```sql
- id (UUID, 主键)
- user_id (UUID, 外键)
- word (字符串，查询的单词/句子)
- translation (字符串，翻译结果)
- created_at (时间戳)
- updated_at (时间戳)
- mastered (布尔，是否掌握)
- review_count (整数，复习次数)
```
