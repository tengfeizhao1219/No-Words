# 不记单词 · 有道翻译版

一个基于有道翻译API的现代化单词翻译工具，支持单词查询、句子翻译、双语对照和词根分析。

## 🚀 立即部署

### 方法一：一键部署（推荐）
```bash
# 给部署脚本执行权限
chmod +x deploy.sh

# 运行部署脚本
./deploy.sh
```

### 方法二：直接使用
1. 下载 `index.html` 文件
2. 用浏览器直接打开
3. 立即开始使用！

### 方法三：在线部署
- **GitHub Pages**: 上传到GitHub仓库并启用Pages
- **Vercel**: 拖拽文件夹到Vercel网站
- **Netlify**: 拖拽文件夹到Netlify网站
